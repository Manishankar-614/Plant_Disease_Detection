About This Project

This web application uses a multi-task deep learning model to classify images of plants.

1. The Model

The model is a MobileNetV2, pre-trained on ImageNet, with two separate classification heads:

  • Part Head: Classifies the plant part (e.g., Fruits, Leaves, Stems, or Not_a_plant).
  • Disease Head: Classifies the specific disease present on the plant part.

2. The Project

This tool was built to demonstrate the deployment of a multi-output Keras/TensorFlow model using a simple, interactive web GUI.

  • Backend: Python, Flask
  • Frontend: HTML, CSS, JavaScript
  • Database: SQLite

3. How it Works

  1. Upload: You provide an image via the "Predict" page.
  2. Preprocess: The image is resized to 224x224 and preprocessed for MobileNetV2.
  3. Predict: The Flask backend runs the model, which results in two separate outputs (part and disease).
  4. Display: The JavaScript on the page receives the JSON result and updates the UI.
  5. Log: Every prediction is saved to a local SQLite database, which you can view on the History tab.
